# ECOBLACK landing — ajustes aplicados

## Cambios principales
- ECOBLACK se comunica como empresa/marca, no como producto.
- El producto se comunica como Carbon Black recuperado en formato briqueta.
- El uso validado por la empresa se enfoca en combustible sólido para hornos y calderas industriales.
- Pigmento, carga industrial y materia prima secundaria se presentan como usos potenciales sujetos a validación técnica.
- Se corrigió la ruta del video loop y se optimizó su carga para hero.
- Se mejoró el contraste del video de fondo, manteniendo legibilidad del texto.
- Se agregó una tarjeta de video dentro de la sección de infraestructura productiva.
- Se agregaron Open Graph, Twitter Card, preload del poster, lazy loading en imágenes y comentarios de integración futura del formulario.

## Formulario
El formulario sigue siendo una demo visual. Para producción puede conectarse a CRM, correo, Google Sheets, API/webhook o backend propio.
